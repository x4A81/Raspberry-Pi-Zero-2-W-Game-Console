# Raspberry-Pi-Zero-2-W-Game-Console
This is a handheld game console based on Raspberry Pi Zero 2 W and RetroPie.
The hardware included are:
- 3.5" LCD display
- Raspberry Pi Zero 2 W
- 3.7V 3000mAh lipo battery
- TP4056 USB-C 1A lipo charger board
- DC-DC boost converter
- I2S PCM5102A DAC with Headphone Jack

To use the game console you first have to flash RetroPie to a microSD card and then insert it into the Pi.
I made this project as I wanted to play some retro games but I didn't want to purchase multiple consoles. So I decided that I wanted to build one!
